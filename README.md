# JiaYu Here

这是 `www.jiayuhere.com` 的静态网站版本，适合部署到 GitHub Pages、Netlify、Vercel 或任意静态托管服务。

## 文件说明

- `index.html`: 网站首页
- `assets/styles.css`: 页面样式
- `assets/script.js`: 移动菜单、文章筛选、订阅表单演示
- `assets/images/hero-travel-desk.png`: 首页封面图
- `CNAME`: 自定义域名配置，适用于 GitHub Pages
- `robots.txt`: 搜索引擎抓取配置
- `sitemap.xml`: 站点地图
- `site.webmanifest`: 浏览器安装与主题信息
- `favicon.svg`: 网站图标

## 上线提示

1. 上传整个文件夹到静态托管平台。
2. 如果使用 GitHub Pages，保留 `CNAME` 文件。
3. 在域名服务商处配置 `www.jiayuhere.com` 指向托管平台提供的地址。
4. 上线后把真实文章链接、邮件订阅服务和统计工具接入即可。
